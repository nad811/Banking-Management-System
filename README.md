Banking Management System in C

A console-based Banking Management System developed using C programming language. The project provides basic banking operations such as account creation and transaction handling using a menu-driven interface.

Features

- Create new bank account
- Display all accounts
- Search account by account number
- Deposit money
- Withdraw money
- Basic balance management
- Persistent data storage using file handling

Technologies Used

- C Programming
- Structures
- Arrays
- Functions
- File Handling

Project Structure

- main.c → Program entry and menu system
- account.c → Core banking operations
- account.h → Function and structure declarations
- file.c → Handles saving and loading data
- file.h → File-related function declarations
- data.dat → Stored account data

Key Learning Outcomes

- Understanding basic banking system logic
- Implementing CRUD operations in C
- Working with file handling for data persistence
- Using modular programming with multiple source files
- Managing structured data using structs
